# MACOS Style Porfolio

## Dependencies | Packages | Libraries used

- Vite as a frontend framework
  - React as a frontend library

## Implementation

1. Create a React app using Vite

   - `npm create vite@latest .`
     _Follow [vite doc](https://vite.dev/guide/) for more information_

2. Add tailwindcss into the project
   - `npm install tailwindcss @tailwindcss/vite`
     _Follow [tailwindcss doc](https://tailwindcss.com/docs/installation/using-vite) for more information_
